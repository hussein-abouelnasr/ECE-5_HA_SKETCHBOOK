Motion Sensing Project


Challenge 1 was my first subproject for the final project and it consisted of building an Ultrasonic motion sensing system that read out live distances to the serial monitor. I then built upon that to create a motion sensing alarm security system. The first image is a skecth of Challenge 1's motion sensing system in action. The second image is a flowchart representing the sequential processes involved in the intruder alarm.


The first image is a sketch of the visual representation of the 1-D rotating room scanner/motion tracker built as part of Challenge 2. The sketch visualizes the area scanned and the process of converting from polar to Cartesian so as to have instance mapped out in x-y coordinates. The second image is a flowchart mapping out the process of Challenge 2 and the line of communiation opened between Arduino and Python through Pyserial.


The second image is both a visual and written representation of Kernel Density Estimation which is the mathematical technique used to analyze the data recorded by the sensor in Challenge 3. The first image is a representation of a few of the different ways in which we can visualize our recorded data, both when it's raw and unfiltered as well as after it has been turned into a pdf using KDE.
